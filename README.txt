The typing game with Concurrency

Running Instructions
"make" to compile into src folder
"make run" to use example_dict.txt

You can change the number of concurrent words in the makefile as well as the toltal words before the game ends by changing
NOWORDS and TOTALWORDS respectively in the makefile.

You can change the dictionary used by changing DICT in the makefile.
